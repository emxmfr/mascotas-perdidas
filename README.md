# Mascotas Perdidas

Tablón comunitario para registrar mascotas perdidas/encontradas, con foto y filtros por tipo, estado y zona.

## Stack
- Next.js (frontend)
- Supabase (base de datos + almacenamiento de fotos)
- Vercel (hosting, gratis)

## Cómo correrlo en tu computadora
1. `npm install`
2. Copia `.env.local.example` a `.env.local` y pon tus datos de Supabase
3. `npm run dev`
4. Abre http://localhost:3000

Instrucciones completas de configuración de Supabase y despliegue en Vercel: sigue la guía que te dio Claude en el chat.
